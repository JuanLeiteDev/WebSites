# Repositório de Estudos e Evolução em HTML e CSS

Este repositório público reúne os meus projetos e exercícios de desenvolvimento web, com foco atual em `HTML` e `CSS`. A ideia é usar este espaço como um registo organizado da minha evolução técnica, acompanhando a forma como estruturo, construo e melhoro cada projeto ao longo do tempo. Futuramente, o repositório poderá também incluir trabalhos com `JavaScript`.

## Sobre o repositório

O principal objetivo deste repositório é documentar a minha progressão em programação através de projetos práticos. Mais do que guardar ficheiros, este espaço funciona como um histórico de aprendizagem, onde procuro manter uma estrutura clara, separação entre conteúdos e consistência na organização dos projetos.

Por ser um repositório público, também serve como uma forma transparente de mostrar o meu percurso, as competências que estou a desenvolver e a maneira como vou evoluindo na área do desenvolvimento web.

## Contexto pessoal

Sou estudante do 1.º ano de `LESI` (`Licenciatura em Engenharia de Sistemas Informáticos`) no `IPCA`, campus de Barcelos. Este repositório foi criado como parte do meu processo de aprendizagem, prática e consolidação de bases em programação, especialmente no contexto do front-end.

## Estrutura atual do projeto

Neste momento, o repositório segue uma organização simples e bem definida, separando os ficheiros por função:

```text
.
├── index.html
├── README.MD
└── src
    ├── assets
    │   ├── css
    │   ├── fonts
    │   ├── img
    │   └── js
    └── site
```

### Organização das pastas

- `index.html`: página principal ou ponto de entrada inicial do repositório.
- `src/site`: ficheiros HTML de cada projeto.
- `src/assets/css`: folhas de estilo separadas por projeto.
- `src/assets/img`: imagens e recursos visuais utilizados nas páginas.
- `src/assets/fonts`: espaço reservado para tipografias personalizadas.
- `src/assets/js`: espaço preparado para futuras implementações com JavaScript.

Esta estrutura permite manter os projetos organizados, facilitar a manutenção e separar claramente conteúdo, estilos e recursos.

## Projetos e estado atual

O repositório já inclui projetos com ficheiros HTML e CSS organizados de forma independente. Um dos exemplos atuais é o projeto **Cordel Moderno**, composto por ficheiros separados para estrutura e estilo, seguindo uma abordagem mais limpa e modular.

À medida que eu for aprendendo novos conceitos e desenvolvendo novas páginas, este repositório será atualizado com mais projetos, melhorias visuais e, possivelmente, novas interações com JavaScript.

## Como visualizar localmente

Para ver os projetos, basta abrir os ficheiros `.html` no navegador. Dependendo do que quiseres consultar:

- abre `index.html` para aceder ao ponto de entrada principal;
- abre ficheiros dentro de `src/site/` para visualizar projetos específicos.
